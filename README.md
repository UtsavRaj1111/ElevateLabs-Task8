# Task 8: K-Means Clustering - Mall Customer Segmentation

## 📌 Objective
Perform **unsupervised learning** using K-Means clustering to segment mall customers based on their spending patterns and annual income.

## 📂 Dataset Description
The **Mall Customers dataset** contains:
- **CustomerID**: Unique identifier
- **Gender**
- **Age**
- **Annual Income (k$)**
- **Spending Score (1-100)**

**Purpose:** Identify distinct customer groups for targeted marketing.

## 🛠 Tools & Libraries
- **Python**
- **Pandas** for data handling
- **Matplotlib & Seaborn** for visualization
- **Scikit-learn** for clustering and evaluation

## 📊 Steps
1. Load and explore the dataset.
2. Preprocess and select relevant features.
3. Standardize features for distance-based clustering.
4. Use the **Elbow Method** to find the optimal number of clusters (K).
5. Train **K-Means** and assign cluster labels.
6. Visualize results (with PCA for 2D view).
7. Evaluate clustering with **Silhouette Score**.

## 📈 Output
- **Cluster visualizations** using PCA
- **WCSS vs. K plot** for Elbow Method
- **Silhouette Score** to assess quality

## 📚 Learning Outcomes
- Understanding **unsupervised learning**
- Applying **K-Means clustering**
- Using **Elbow Method** and **Silhouette Score** for model evaluation
- Data visualization for cluster interpretation
